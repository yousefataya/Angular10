Description

The ColorPicker component is a user interface to select and adjust color values. This supports various color specifications like RGB (Red Green Blue), HSV (Hue Saturation Value), and Hex codes. In this sample, Select the shirt color from different customized palettes. It contains circle, square, rounded edge, and multiple scroll palettes. Using the presetColors property, you can specify the custom colors to be loaded. More information about ColorPicker can be found in this documentation section.
