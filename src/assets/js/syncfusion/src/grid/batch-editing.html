This sample demonstrates CRUD operations in Grid. You can perform CRUD operations as follows,

  • Add - To add new record, click Add toolbar button
  • Edit - To edit record, double click a cell
  • Delete - To delete record, click toolbar Delete button after selected a row
  • Update,Cancel - You can save or discard changes by click toolbar Update and Cancel button respectively

Batch editing mode is used to send bulk request (add, update and delete) to Grid. You can start editing by double clicking a cell and can change the cell value. Edited cell will be highlighted while navigating to next cell or any other row, so that you know which fields or cells has been edited.

In this demo, batch editing is enabled by defining mode as `batch`.
