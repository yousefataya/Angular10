This sample demonstrates the Grid searching feature. In this sample, use the search box from toolbar to search Grid records.

The Grid has an option to search its content using the search method with search key as the parameter.

In this demo, The Grid toolbar provides an option to search the Grid's records. The user can type the text box in the toolbar and click search button or press Enter key to perform search operation. We have used search method to initiate search.

More information on the searching configuration can be found in this documentation section.
