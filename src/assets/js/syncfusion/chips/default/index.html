Description

Chip is a small block of essential information that triggers an event on click action. It contains the text, an image, or both, mostly used in contacts, mails or filter tags. In this sample, default, input, filter and choice chips are demonstrated with their default functionalities. Input chip– Basic chip with delete icon, enables to remove chips from the chip list collection. Choice chip – Used to choose a choice from available options. Filter chip – Used to select multiple choices from available options.
