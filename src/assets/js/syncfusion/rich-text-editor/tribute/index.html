Description

The Rich Text Editor allows you to integrate the third-party libraries such as Tribute JS or At JS for mentions. If you want to tag or address someone directly during a conversation, the mentions library will be helpful. Type @ to show autocomplete popup with a list of matching items based on the provided key-value pair. For example, type '@' in the above sample to show the employee list with their mail IDs in the autocomplete popup.
