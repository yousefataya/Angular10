Description

ButtonGroup is a graphical user interface that groups series of buttons horizontally or vertically. This supports radio and checkbox type behaviors. The above sample demonstrates the behaviors of button groups with icon only, text only, and text with icon combinations. Default: Triggers action on button click. Single selection: Radio type behavior selects a single button and submits its value to the server on form submission. This is showcased with the second button disabled, by default. Multiple selection: Checkbox type behavior selects multiple buttons and submits its selected values to the server on form submission. This is showcased with the first button selected, by default. More information on ButtonGroup can be found in this documentation section.
