Description

The ComboBox has been provided with several options to customize each list items, group title, header and footer elements. This sample uses the following list of templates in ComboBox ItemTemplate - To customize the list item's content. HeaderTemplate - To customize the header element. More information on the template feature configuration can be found in the documentation section.
