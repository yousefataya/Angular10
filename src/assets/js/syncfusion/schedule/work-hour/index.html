Description

In this demo, the work hours are set by using the workHours property and it usually applies only on the working days defined for the Scheduler. It can either be highlighted or not by defining the highlight option within the workHours property. The working hour range can be defined by making use of the start and end option available within the workHours property. To set discontinuous working hours on a day, then the default workHours on Scheduler needs to be disabled by setting false to highlight option within it. Then, make use of the setWorkHours method which accepts the days collection and the start & end hour values as parameters.
